# How To Use
It's simple:
```
**Linux**

# Clone This Repository
https://github.com/GenXPS/ImGroot.git

# Go Into The Repository
cd ImGroot

# Permission Acces
chmod +x Main_ImGroot

# Run The App
./Main_ImGroot
```
```
**Windows**

# Clone This Repository
https://github.com/GenXPS/ImGroot.git
Download ZIP

# Go Into The Repository
extract Files

# Permission Access
g++ -o ImGroot Main_ImGroot.cpp

# Run The App
ImGroot.exe
```
